## Django项目创建和启动
### PyCharm创建
使用Django模板即可，建议下好Django后创建项目，选Conda创建就行，最好别总是创建虚拟环境。
### PyCharm启动
配置运行的基本配置即可运行，很方便。
### 命令行创建
```text
python manage.py startapp 应用名
```
### 命令行运行
```text
python manage.py runserver 
manage.py runserver 80      # 指定端口
```
